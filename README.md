# all_things
Дипломная работа посвящена моделированию многочастичных систем. Файл project.ipynb содержит код программы, моделюрующий движение молекул в различных системах. В программе присутствуют гибкие настройки взаимодействия молекул и многое другое.  При запуске кода вы увидете анимацию движения молекул в трехмерном случае.
